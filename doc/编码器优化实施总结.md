## 🛠️ 实施细节

### 1. 新增模块（位于 src/models/encoder.py）

#### SincConv1d (L856-L968)
- 可学习带通滤波器，仅学习截止频率
- Mel-scale 初始化覆盖心音频段（25-400Hz）
- 输出绝对值作为能量检测

#### AttentiveStatisticsPooling (L973-L1034)
- 提取关键帧的加权均值和方差
- 输出拼接 [mean, std]，保留更多统计信息

#### DepthwiseSeparableConv1d (L1039-L1096)
- 深度可分离卷积，大幅减少参数
- 支持 GroupNorm 和 BatchNorm

#### SincNetECAMambaEncoder (L1101-L1312)
- 6个 Stage 的完整编码器
- 与 CNNMambaEncoder 接口完全兼容

### 2. 工厂函数更新（L1317-L1336）
```python
build_encoder("sincnet_eca_mamba", ...)
```

### 3. PAHCLModel 适配（src/models/pahcl.py）
- 自动识别新编码器类型
- 正确设置 encoder_out_dim=192
- 支持 MoCo 动量编码器

### 4. 配置文件更新（configs/pretrain.yaml）
- 添加新编码器配置说明
- 提供完整使用示例

## 📝 使用方法

### 快速开始

**1. 修改配置文件**

编辑 `configs/pretrain.yaml`:
```yaml
model:
  encoder_type: "sincnet_eca_mamba"  # 改为新编码器
  mamba_d_model: 128                  # 降低维度
  mamba_n_layers: 2                   # 减少层数
  pool_type: "asp"                    # 使用 ASP 池化
```

**2. 运行预训练**
```bash
python scripts/pretrain.py --config configs/pretrain.yaml
```

**3. 下游微调**
```bash
python scripts/finetune.py \
    --task circor_murmur \
    --pretrained checkpoints/pretrain/best_model.pt
```

### 完整配置示例
```yaml
model:
  encoder_type: "sincnet_eca_mamba"
  
  # SincNet
  sinc_out_channels: 64
  sinc_kernel_size: 251
  sinc_stride: 2
  
  # 阶段配置
  stage1_channels: [64, 96]
  stage2_channels: [96, 128]
  
  # Mamba
  mamba_d_model: 128
  mamba_n_layers: 2
  mamba_d_state: 16
  mamba_d_conv: 4
  mamba_expand: 2
  mamba_dropout: 0.1
  
  # 其他
  use_groupnorm: true
  num_groups: 8
  eca_kernel_size: 3
  cycle_output_dim: 192
  pool_type: "asp"
```

## 🔬 技术亮点

### 1. SincNet 可解释性
- 学习的滤波器频率可视化
- 初始化覆盖心音关键频段（25-400Hz）
- 前5个滤波器示例：
  ```
  滤波器 0: 50.0 - 79.7 Hz  (带宽: 29.7 Hz)
  滤波器 1: 54.7 - 84.5 Hz  (带宽: 29.8 Hz)
  滤波器 2: 59.5 - 89.3 Hz  (带宽: 29.8 Hz)
  滤波器 3: 64.3 - 94.1 Hz  (带宽: 29.8 Hz)
  滤波器 4: 69.1 - 99.0 Hz  (带宽: 29.9 Hz)
  ```

### 2. 轻量化设计
- 深度可分离卷积减少参数
- 2层 Mamba 平衡性能和效率
- GroupNorm 对小批量更稳定

### 3. 统计池化
- ASP 提取关键帧信息
- 同时保留均值和方差
- 输出 256维 → Linear → 192维

### 4. 接口兼容
- 与 CNNMambaEncoder 接口完全一致
- 即插即用，无需修改训练代码
- 支持所有现有功能（MoCo、子结构对比等）
