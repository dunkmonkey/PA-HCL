## 🚀 快速开始命令

### 预训练

```bash
# 使用新编码器（推荐）
python scripts/pretrain.py --config configs/pretrain.yaml

# 使用旧编码器
python scripts/pretrain.py --config configs/pretrain.yaml
```

### 微调

```bash
# 自动检测编码器类型
python scripts/finetune.py \
    --task circor_murmur \
    --pretrained checkpoints/pahcl_sincnet_pretrain/best_model.pt

# 线性评估
python scripts/finetune.py \
    --task circor_murmur \
    --pretrained checkpoints/pahcl_sincnet_pretrain/best_model.pt \
    --linear-eval
```

### 监督基线

```bash
# 使用新编码器
python scripts/train_supervised_baseline.py \
    --task physionet2016 \
    --encoder-type sincnet_eca_mamba

# 使用旧编码器
python scripts/train_supervised_baseline.py \
    --task physionet2016 \
    --encoder-type cnn_mamba
```

## 🔧 按场景选择配置

### 场景1: 快速实验（资源有限）
**推荐**: `pretrain.yaml`

**优势**:
- 参数量少 83%
- 训练速度快 2-3倍
- 显存占用小 40%

**配置**:
```yaml
encoder_type: "sincnet_eca_mamba"
batch_size: 64
gradient_accumulation_steps: 4
```

---

**方案**: 同时测试两种编码器

**新编码器** - `pretrain.yaml`:
```yaml

```

**旧编码器** - `pretrain.yaml`:

## 📝 配置修改要点

### 1. 切换到新编码器

**在 pretrain.yaml 中修改**:
```yaml
model:
  encoder_type: "sincnet_eca_mamba"  # 改这里
  mamba_d_model: 128                  # 改这里
  mamba_n_layers: 2                   # 改这里
  pool_type: "asp"                    # 改这里
```

**或者直接使用**:
```bash
python scripts/pretrain.py --config configs/pretrain.yaml
```

---

### 2. 微调时匹配编码器

**自动匹配（推荐）**:
```bash
# 脚本会自动从检查点读取编码器类型
python scripts/finetune.py --task circor_murmur --pretrained xxx.pt
```

**手动指定（不推荐）**:
```yaml
# 在任务配置文件中
model:
  encoder_type: "sincnet_eca_mamba"
  mamba_d_model: 128
  mamba_n_layers: 2
```

---

### 3. 调整批量大小

由于新编码器更轻量，可以使用更大的批量:

```yaml
training:
  batch_size: 96  # 从 64 增加到 96
  gradient_accumulation_steps: 4  # 有效 batch = 384
```

## ⚠️ 常见错误

### 错误1: 编码器类型不匹配
```
AssertionError: checkpoint encoder_type mismatch
```

**解决**: 
- 检查预训练检查点的编码器类型
- 微调配置必须与预训练一致

---

### 错误2: 维度不匹配
```
RuntimeError: size mismatch for cycle_projector.mlp.0.weight
```

**解决**:
- 新编码器输出 192 维
- 旧编码器输出 256 维
- 不能混用检查点

---

### 错误3: 缺少配置参数
```
AttributeError: 'Namespace' object has no attribute 'sinc_out_channels'
```

**解决**:
- 使用 `pretrain.yaml` 或
- 在配置中添加所有新编码器参数
